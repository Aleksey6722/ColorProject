ColorsProject

Проект ColorsProject представляет собой веб-приложение, реализующее функцию поиска и подбора автомобиля (или нескольких) по цвету, введённому пользователем. Для этого, пользователю предоставлен интерфейс для выбора цвета и желаемого количества автомобилей. Для зарегистрированных пользователей есть возможность сохранять автомобили в персональном разделе "Избранное".

Немного технической информации

Веб-приложение проекта ColorsProject написано на языке программирования Python. Python отличается удобством и лаконичностью синтаксиса, множеством встроенных и внешних библиотек, широкой применяемостью в разных направлениях IT. Это делает его одним из самых популярных языков для бэкенд-разработки. В частности, на Python написано множество фреймворков для разработки веб-приложений. ColorsProject создан на популярном фреймворке Django, что позволяет без проблем отлаживать, масштабировать и наращивать проект дополнительным функционалом

Приложение имеет API уровень (программный интерфейс), отвечающий за обработку поступающих данных, связь и работу с базой данных и формирование ответа сервера. Программный интерфейс позволяет обеспечить доступ к функционалу приложения сторонним клиентам. API проекта написан с использованием расширения Django Rest Framework - удобным инструментом для создания RESTfull API.

Неотъемлемой частью приложения является работа с базой данных, содержащей информацию о зарегистрированных пользователях, ключи сессий, информацию об автомобилях. Ниже представлена схема, отображающая реляционные связи между всеми таблицами базы данных

Работа с базой данных осуществляется через систему управления базами данных PostgreSQL. Django предоставляет технологию объектно-реляционного отображения для работы с базами данных - DjangoORM. В приложении ColorsProject используются механизмы этой ORM для чтения, создания и изменения записей в базе данных

Фронтенд часть

Фронтенд часть представляет собой сайт, состоящий из нескольких HTML страниц: главная страница, страница о проекте и "Избранное". Доступ к последней имеют только зарегистрированные пользователи. Регистрация и аутентификация осуществляются через соответствующие формы. Данные полей каждой формы проходят валидацию на сервере и только после её успешного прохождения пользователь получает доступ. Навигация по сайту осуществляется через меню, расположенное в шапке. Динамические функции на стороне клиента осуществляются через скрипты, написанные на языке JavaScript с использованием библиотеки JQuery. В частности, скрипты отвечают за отображение/скрытие паролей в формах, изменение стилей кнопок меню при нажатии на них и другие функции. Важнейшей задачей, которую выполняются скрипты, является отправление запросов на сервер и обработка ответов. Так как, сервер предоставляет API, общение между ним и клиентом осуществляется через отправку данных в JSON формате. Для реализации такой связи между клиентом и сервером используется технология AJAX. Скрипты, применяющие данную технологию отправляют HTTP-запрос с необходимыми данными на API сервера, получают и обрабатывают ответ. В зависимости от кода состояния ответа, определяется поведение клиентского приложения - например отображаются сообщения об ошибке, скрываются/отображаются кнопки, осуществляется перенаправление пользователя на другую страницу и т.п. В рамках проекта ColorsProject вышеописанная технология применяется для отправки на сервер данных форм аутентификации и регистрации, данных о выбранном цвете и количестве машин. На основе ответов сервера определяется поведение клиента. Таким образом, в основе фронтенд части проекта лежат технологии HTML, CSS, AJAX и язык JavaScript.

Подводем итог. Проект ColorsProject написан с использованием актуальных технологий программирования; обладает потенциалом для маштабирования и расширения функционала; имеет перспективу применения в качестве сервиса для сайтов продажи автомобилей, интернет-магазинов, сайтов автосалонов и на других направлениях, связанный с автомобильной тематикой.
